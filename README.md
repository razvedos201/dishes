# Мои блюда (Flutter / Android)

Локальное приложение для хранения блюд и их рецептов с возможностью шаринга через мессенджеры и экспорта/импорта в JSON.

## Возможности

- ✅ Добавление блюд с произвольным числом ингредиентов
- ✅ Вес ингредиентов: граммы по умолчанию, можно выбрать килограммы
- ✅ Необязательное поле стоимости ингредиента + автоматический подсчёт итоговой стоимости блюда
- ✅ Картинка блюда (камера или галерея), хранится локально
- ✅ Локальное хранение всех данных на устройстве (без сервера)
- ✅ Список блюд на главном экране, тап → детальная карточка
- ✅ Кнопка «Отправить» — шаринг через любые мессенджеры (текст + картинка)
- ✅ Экспорт всех блюд в JSON-файл (для бэкапа / переноса)
- ✅ Импорт JSON-файла (с выбором: добавить или заменить)
- ✅ Редактирование и удаление блюд

## Структура проекта

```
lib/
├── main.dart                       # Точка входа, тема приложения
├── models/
│   ├── dish.dart                   # Модель блюда + сериализация JSON
│   └── ingredient.dart             # Модель ингредиента
├── services/
│   └── storage_service.dart        # Запись/чтение, экспорт/импорт, картинки

    ├── home_screen.dart            # Список блюд + меню экспорта/импорта
    ├── dish_edit_screen.dart       # Создание/редактирование блюда

```

## Запуск

Требуется установленный [Flutter SDK](https://docs.flutter.dev/get-started/install) (3.10+).

```bash
# 1. Установить зависимости
flutter pub get

# 2. Подключить Android-устройство (с включённой отладкой по USB) или запустить эмулятор
flutter devices

# 3. Запустить
flutter run

# Собрать релизный APK для установки на телефон
flutter build apk --release
# Готовый файл: build/app/outputs/flutter-apk/app-release.apk
```

## Где хранятся данные

Все блюда сохраняются в `getApplicationDocumentsDirectory()` (внутреннее хранилище приложения):

- `dishes.json` — список всех блюд
- `dish_images/` — папка с картинками блюд

При удалении приложения данные удаляются. Для бэкапа используйте кнопку **Экспорт в JSON** в меню (три точки в правом верхнем углу).

## Заметки

- При экспорте JSON картинки **не включаются** (только пути), потому что пути не имеют смысла на другом устройстве. Сами блюда и ингредиенты переносятся полностью. Картинки добавляются заново после импорта.
- Вес ингредиентов внутри JSON всегда хранится в граммах для единообразия. На экране редактирования можно вводить и в кг — приложение конвертирует автоматически.
- iOS не настроена (по требованию — пока только Android). Если потребуется, нужно добавить в `ios/Runner/Info.plist` разрешения `NSCameraUsageDescription` и `NSPhotoLibraryUsageDescription`.
